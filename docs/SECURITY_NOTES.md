# Security and Privacy Notes

Before publishing, this repository was prepared as a clean subset of the working directory.

Included:

- reproduction code
- helper scripts
- small result images and JSON summaries
- dataset download links
- experiment notes

Excluded:

- raw datasets
- dataset archives
- SSH commands
- server addresses
- passwords
- access tokens
- private environment files
- virtual environments
- full training checkpoints

The repository `.gitignore` also blocks common credential patterns such as `.env`, private key files, and token/password-named files.
