# Result Artifacts

This directory contains small result artifacts from smoke, tiny-overfit, and
SemanticKITTI sequence `08` validation benchmark runs.

Included artifact types:

- `summary.json`
- `OVERFIT_NOTES.md`
- loss curves
- BEV ground-truth and prediction images
- LiDAR/image projection sanity checks
- IPFP extra-point projection sanity checks
- full-frame validation metric JSON files and selected validation montages

Raw datasets, extracted point-cloud files, and checkpoints are not included.
