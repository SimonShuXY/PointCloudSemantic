# Result Artifacts

This directory contains small result artifacts from smoke and tiny-overfit runs.

Included artifact types:

- `summary.json`
- `OVERFIT_NOTES.md`
- loss curves
- BEV ground-truth and prediction images
- LiDAR/image projection sanity checks
- IPFP extra-point projection sanity checks

Raw datasets, extracted point-cloud files, and checkpoints are not included.
